계정 및 데이터 삭제 안내

이 페이지는 사용자가 앱에서 생성한 계정과 관련 데이터를 삭제 요청할 수 있는 절차를 안내합니다.

🔐 계정 삭제 방법

  1. 앱 실행 후, 설정 → 계정 관리 화면으로 이동합니다.
  2. “계정 삭제 요청” 버튼을 눌러 삭제를 신청합니다.
  3. 확인 창에서 다시 한 번 삭제 여부를 확인하면 즉시 처리됩니다.

🗂 삭제되는 데이터

  • 사용자 계정 정보 (이메일, 닉네임 등)
  • 앱 내부에 저장된 사용자 기록
  • 서버에 저장된 개인 데이터

⏱ 데이터 보관 기간

법적 요구 사항이 없는 경우 사용자의 삭제 요청 즉시 모든 데이터는 완전 삭제됩니다.
